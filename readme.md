# Deanteria Fantasy Map

This project is a static interactive fantasy map built for GitHub Pages.

## Files

- `index.html` — main page and map container
- `styles.css` — styling for map, controls, markers, and overlays
- `script.js` — pan/zoom logic, marker rendering, and zoom-level behavior
- `assets/map.svg` — placeholder map image
- `assets/icons/` — marker icon graphics

## Usage

Open `index.html` in a browser to test locally.

## Deploy to GitHub Pages

1. Push this repository to GitHub.
2. In repository Settings, enable GitHub Pages for the `main` branch and root folder.
3. Visit the published site URL to see the map.
